# OpenClaw 安全卫士 - 设计文档

## 项目概述

**产品名称**: OpenClaw Guard (OpenClaw 安全卫士)
**目标平台**: macOS
**技术栈**: Go + Vue 3 + Electron
**产品定位**: 轻量级 AI 编程助手安全检测与监控工具

---

## 核心功能

### 1. 一键安全扫描
- 检测 OpenClaw/Claude Code 配置文件
- 扫描敏感信息泄露（API Key、密钥等）
- 检查文件权限和安全配置
- 分析历史操作日志中的风险
- 生成安全报告和优化建议

### 2. 实时监控告警
- 监控文件操作（读/写/删除）
- 检测危险命令执行
- 路径白名单控制
- 弹窗确认高风险操作

## IPC 通信协议

### 请求格式
```json
{
  "id": "unique-request-id",
  "action": "scan|monitor|config",
  "data": {}
}
```

### 响应格式
```json
{
  "id": "unique-request-id",
  "success": true,
  "data": {}
}
```

### 事件推送
```json
{
  "event": "alert|status",
  "data": {}
}
```

## 检测规则

### 配置扫描规则
- API Key 明文检测
- 敏感路径暴露检查
- 文件权限审查
- 历史日志审计

### 监控拦截规则
- 危险路径黑名单
- 危险命令模式
- 敏感文件操作
- 网络请求检测

## 构建流程

1. Go 后端编译为二进制
2. Vue 前端打包
3. Electron 打包整合
4. 生成 .dmg 安装包
