# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please **DO NOT** open a public issue. Instead, please report it via one of the following methods:

1. **GitHub Security Advisory**: Use the "Report a vulnerability" button on the GitHub Security tab.
2. **Email**: (Maintainer to add email here)

We take all security reports seriously and will acknowledge your report within 48 hours.

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

# 安全政策

## 报告漏洞

如果您在本项目中发现安全漏洞，请**不要**公开发布 Issue。请通过以下方式报告：

1. **GitHub 安全通告**：使用 GitHub 项目 "Security" 标签页下的 "Report a vulnerability" 按钮。
2. **私密渠道**：请联系维护者。

我们将认真对待所有报告，并在 48 小时内给予回复。
