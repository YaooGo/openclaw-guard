# OpenClaw 安全卫士

一款为 AI 编程助手（OpenClaw、Claude Code 等）设计的安全检测与监控工具。

> [!WARNING]
> **安全免责声明**：本工具仅作为安全辅助手段。在使用 AI 编程助手时，请务必保持警惕，不要在未经审核的情况下执行 AI 生成的高风险指令。开发者不对因使用本工具或 AI 生成指令而导致的任何直接或间接损失负责。

## 功能特性

### 1. 一键安全扫描
- 配置文件安全检查
- 敏感信息泄露检测（API Key、密钥等）
- 文件权限审查
- 历史操作审计
- 生成安全报告

### 2. 实时监控告警
- 文件操作监控（读/写/删除）
- 危险命令检测
- 路径白名单控制
- 弹窗确认高风险操作

### 3. 配置管理
- 自定义路径白名单/黑名单
- 配置危险命令检测规则
- 敏感度级别调整

## 开发环境

### 环境要求
- Go 1.23+
- Node.js 18+
- npm/yarn/pnpm

### 安装依赖

```bash
# 后端依赖
cd backend
go mod download

# 前端依赖
cd ../frontend
npm install

# Electron 依赖
cd ../electron
npm install
```

### 开发运行

```bash
# 1. 启动前端开发服务器（在 frontend 目录）
npm run dev

# 2. 启动 Electron（在 electron 目录，设置 NODE_ENV=development）
NODE_ENV=development npm start
```

### 构建生产版本

```bash
# 1. 构建前端
cd electron
npm run build

# 2. 编译 Go 后端
npm run build-go

# 3. 打包 Electron 应用
npm run dist:mac
```

## 使用说明

1. **首次使用**：打开应用后，点击"开始扫描"进行安全检查
2. **查看结果**：扫描完成后查看风险详情和优化建议
3. **实时监控**：在"实时监控"页面查看操作日志和拦截记录
4. **配置管理**：在"设置"页面自定义白名单和检测规则

## 安全规则

### 默认黑名单路径
- `~/.ssh` - SSH 密钥目录
- `~/.gnupg` - GPG 密钥目录
- `/etc` - 系统配置目录
- `/usr` - 系统程序目录
- `/System` - macOS 系统目录

### 默认白名单路径
- `~/Projects` - 项目目录
- `~/code` - 代码目录
- `~/Documents` - 文档目录
- `~/Desktop` - 桌面目录

### 危险命令检测
- `rm -rf` - 强制删除
- `rm -r /` - 删除根目录
- `mkfs` - 格式化文件系统
- `dd if=` - 磁盘写入
- `format` - Windows 格式化
- 等等...
